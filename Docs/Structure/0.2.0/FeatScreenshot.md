> 2018-07-27 문서 작성

## 問題

OPGS16ならではのスクリーンショット機能を実装する必要があると思う。`0.1.0`では実装しなかったが一々ユーザーがAlt+PrtScnで間接的にスクリーンショットを撮ることが面倒かもしれないし、それでスクリーンショットを撮る機能を実装しようとしている。

## 提案

스크린 샷을 찍는 기능을 실장하되, 스크린 샷을 찍는 방법에 대해서는 유저가 알아서 구현을 하도록 스크린 샷 찍기 API 만 제공을 할 것.

스크린 샷 확장자는 `.png` 로 제공을 할 것.
